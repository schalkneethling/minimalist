Minimalist

A minimal pure CSS starter library for most web projects.

Typography

The typography scale is based on a 1.250 (Major third) scale for viewports smaller than 1024 pixels and 1.333 (Perfect fourth) scale for viewport sizes above 1023 pixels. For the font families, Minimalist uses a serif family for headings and a sans-serif family for all other copy. The only exception is for code which uses a monospaced font.

The font stack used is based in typography commonly found preinstalled on most operating systems with appropriate fallbacks. To learn more about the font stack, please visit the system font stack website.

Steps to make pizza dough

  1. Combine flour, salt, and yeast in a large bowl.
  2. Add water and stir until a shaggy dough forms.
  3. Cover bowl with plastic wrap and let rest at room temperature for 8-24 hours.
  4. Transfer dough to a floured work surface.
  5. Divide dough into 4 equal parts.
  6. Shape each part into a round and let rest for 1 hour.

Colors

Minimalist does not provide a color scheme or full palette but does provide a set of neutral colors and a set of colors commonly used for user interface elements such as success, warning and error messages and statuses. All colors are specified in hexadecimal format.
